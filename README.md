## üé® Customization Guide

### 1. Replace Photos

Add your own photos to the `images` folder. You need 6 photos named:
- `photo1.jpg`
- `photo2.jpg`
- `photo3.jpg`
- `photo4.jpg`
- `photo5.jpg`
- `photo6.jpg`

**Tip:** Use high-quality images (at least 1000x1000px) for best results!

### 2. Personalize the Text

Open `index.html` and customize:

- **Line 7:** Change the page title
- **Line 21-24:** Update the anniversary dates and message
- **Line 45-75:** Customize the timeline events with your own memories
- **Line 130-145:** Personalize the love letter (replace `[His Name]` and `[Your Name]`)
- **Line 155-200:** Update the "365 Reasons Why" cards with your own reasons

### 3. Adjust Colors (Optional)

Open `style.css` and modify the color variables at the top (lines 10-17) to match your preferred color scheme.

## üöÄ Deployment to Vercel

### Method 1: Using Vercel CLI (Recommended)

1. Install Vercel CLI:
   ```bash
   npm install -g vercel
   ```

2. Navigate to your project folder:
   ```bash
   cd c:\Users\gaila\OneDrive\Desktop\milsy
   ```

3. Deploy:
   ```bash
   vercel
   ```

4. Follow the prompts and your site will be live!

### Method 2: Using GitHub + Vercel

1. Create a new repository on GitHub
2. Push your code to GitHub:
   ```bash
   git init
   git add .
   git commit -m "Anniversary website"
   git branch -M main
   git remote add origin YOUR_GITHUB_REPO_URL
   git push -u origin main
   ```

3. Go to [vercel.com](https://vercel.com)
4. Click "New Project"
5. Import your GitHub repository
6. Click "Deploy"

Your website will be live in seconds! üéâ
